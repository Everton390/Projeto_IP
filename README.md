Configuração Inicial do Projeto

Este guia detalha como configurar e ativar o ambiente virtual (`venv`) para garantir que o projeto funcione com suas dependências isoladas.

## Pré-requisitos

Certifique-se de ter o **Python** instalado em sua máquina.

---

## 1. Criar o Ambiente Virtual

Execute o comando abaixo no terminal, na pasta raiz do projeto, para criar um ambiente virtual chamado `venv`:

- Crie um ambiente virutal na RAÍZ do diretório:
```bash
python -m venv venv
```

- Ative a venv usando o seguinte comando:

No windows:
```bash
python .\venv\Scripts\activate
```

No Linux/MacOS:
```bash
source venv/bin/activate
```
